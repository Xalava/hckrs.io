
Console commands for connecting to the mongoHQ database. 
(including backup & restore)


--- STAGING APP --- app20276747

[ database url ]
MONGOHQ_URL: mongodb://heroku:64fb61dd02abec6de871330102fb582e@dharma.mongohq.com:10012/app20276747

[ connect to console ]
$ mongo dharma.mongohq.com:10012/app20276747 -u heroku -p 64fb61dd02abec6de871330102fb582e

[ database backup ]
$ mongodump -h dharma.mongohq.com:10012 -d app20276747 -u heroku -p 64fb61dd02abec6de871330102fb582e -o /Users/dev/dev/projects/dev/hckrs/backup/mongohq

[ restore database ]
$ mongorestore -h dharma.mongohq.com:10012 -d app20276747 -u heroku -p 64fb61dd02abec6de871330102fb582e /Users/dev/dev/projects/dev/hckrs/backup/mongohq/app20276747
