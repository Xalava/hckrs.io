Hieronder een uitleg hoe te werken met Meteor.

Regels die starten met het $-teken voer je uit in de Terminal.
Het $-teken zelf is geen onderdeel van het commando, laat dat dus weg!


METEOR INSTALLEREN (éénmalig)

$ curl https://install.meteor.com | sh


METEORITE INSTALLEREN (éénmalig)

1. Install nodejs (including npm)
http://nodejs.org/download/

2. Install Meteorite
$ sudo -H npm install -g meteorite


NAVIGEER NAAR PROJECT

Navigeer naar de projectfolder via de Terminal.
Pas het path aan naar de locatie van de projectfolder op jouw computer.

$ cd ~/Jarno/projects/hckrs


LOKALE SERVER STARTEN

$ mrt run


LOKALE SERVER STARTEN + settings file

$ mrt run --settings settings-local.json


LOKALE SERVER STARTEN + gebruik domein hckrs.io (need DNS configuration)

$ ROOT_URL="http://local1.hckrs.io/" mrt run --settings settings-local.json


LOKALE SERVER STARTEN + aangepaste port

$ ROOT_URL="http://local1.hckrs.io/" mrt run --settings settings-local.json --port 4000


BEKIJK IN BROWSER

Bekijk het resultaat in de browser, via http://localhost:3000 of http://local1.hckrs.io
