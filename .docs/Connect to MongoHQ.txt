
Console commands for connecting to the mongoHQ database. (including backup & restore)
You can also use a (GUI) mongo client (see: http://robomongo.org/)


--- LOCAL APP ---

[ details ]

port:		3002

[ connect to console ]
1. run meteor project
2. open new terminal window
3. navigate to project folder
$ meteor mongo
